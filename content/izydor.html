Izydor

This project is the first of my bachelor at Bellecour Ecole. I also used this project to try the creation of an editor using Rust and EFrame. So, you can create your own game based on it. You have to go at GitHub Releases, to find the last version of the client and the editor. The guide is available here.

Project Requirements

  • Only C/C++.
  • Only Console for rendering.
  • Collaborate with three game designers.
  • Making a story that is not too long.

How to play

You can play it online or using the native version.

Web

On the web version you must click on the buttons.

Native

Those keys suppose that you are using an AZERTY (French) keyboard.

Key Action
Up Arrow Select the choice upper the current
Down Arrow Select the choice under the current
Escape Exit the game
+ Increase the frame size
- Decrease the frame size

Play

So we created a game where you play as Izydor during a Zombie apocalypse in Poland. You can try it on web, or if you are on Windows, you can download it.

Your malware protection and Windows will warn you about the executable.
