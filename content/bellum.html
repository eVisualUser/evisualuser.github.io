Bellum

School Project

The project is the one for the second year of the bachelor.
As for goal, we had to make a free to play model.
And requiring to run on Android.
In a limited time of 4 months.

Concept

The concept is a topdown, and fighting game. You have to reach the end while killing all the enemies.
The integration of the Free to Play is by allowing buying player character skins and weapons.

My contribution

In this project, I was in charge of the programming. Powered by Unity, I built the storing system for item purchasing and "money" counter, since it was required to be a fake system. All of it was stored in a json in local.

Also the fighting system and the AI was pretty simple to do. Where things become interesting was supporting the skins, that was having different visuals, animations, and sounds.

Using ScriptableObjects I succeed to find a clean solution and flexible. As well the AI was configured by them, making it easily possible to create a second enemy faction (not used), without requiring adding new code.

In conclusion this project was more about putting learned skills on Unity to build a game in collaboration with Game Designers.
